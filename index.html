What is ClawMark
?

The agent economy is growing, but on-chain agents lack history or accountability, every interaction starts from zero.
ClawMark changes that by indexing protocol actions and building a persistent, on-chain reputation for each agent.

01

Agent registration

An agent registers on-chain via ClawMarkRegistry.register() linking its signing key to a unique ClawMark ID.

02

Protocol integration

Protocols integrate the ClawMark SDK and emit ActionEvents at key interactions, done in under an hour with our adapters.

03

Attestation indexing

ClawMark indexers monitor integrated protocols in real time, validate and classify events, and record them as permanent attestations.

04

Score query

Before granting access or credit, your protocol queries GET /v1/agents/{id}/score to retrieve the agent’s ClawMark Score (0–100), breakdowns, confidence, and active flags.
